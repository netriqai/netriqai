Key Principles

Typography-Led

Syne for bold headlines (already loaded). Inter for body. Let the type do the heavy lifting — no gradients on text, no glow effects.

Whitespace as Design

Generous padding, clear section breaks. Sections breathe. Content density goes down, comprehension goes up.

Subtle Motion Only

No particles, no floating, no neural-pulse. Micro-interactions: hover lifts, smooth fades on scroll entry, button state transitions. Performance-first.

Strategic Color

Warm off-white base (#FAFAF9). Indigo (#6366F1) as primary accent. Cyan (#06B6D4) as secondary. Dark sections for hero + CTAs only.
