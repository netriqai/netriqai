How should the AI Audit experience feel?

The audit goes from 2 fields → adaptive conversation (3 base + 2-3 branching questions) → AU-contextualized insights

A

Step-by-Step Wizard

One question per screen, with a progress bar. Each answer animates out, next question slides in. Feels focused and guided — like a consultation, not a form. Final screen reveals the personalized report with a "reveal" animation.

Flow: Industry → Business type → State/Region → Team size → Biggest pain point → [Branch: 2-3 contextual follow-ups] → Loading analysis → Report
B

Chat-Style Conversation

Looks like a chat interface. "NeuralShift AI" asks questions one at a time, user responds via buttons/dropdowns (not free text). Messages stack up like a conversation. Creates the feeling of talking to an AI consultant. Report appears as the final "message."

Flow: Chat bubble greeting → Question bubbles with button responses → Typing indicator → Analysis → Report card
C

Smart Form with Live Preview

All core questions visible on the left. As the user fills them in, a "live preview" panel on the right starts building their report in real-time — showing which insights are unlocking based on their answers. Branching questions appear inline when triggered. Feels interactive and immediate.

Flow: Split layout: Questions (left) ↔ Live insight preview (right) → Submit → Full report
